# Function: hexToUint8()

```ts
function hexToUint8(input): Uint8Array
```

Converts a hex string to a uint8 array.

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `input` | `string` | Hex encoded string. |

## Returns

`Uint8Array`

Uint8 array corresponding to the input.
